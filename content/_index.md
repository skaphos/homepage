title: "Skaphos"
---

Skaphos is an operational control system that helps platform engineers run Kubernetes with explicit state, deterministic behavior, and operational clarity.

We build focused, Kubernetes-native tools for platform teams that need deliberate control of infrastructure operations. Rather than abstracting Kubernetes away, Skaphos works directly with Kubernetes primitives and APIs to keep platform state explicit, verifiable, and predictable.

## What guides the work

- Explicit state over implicit behavior.
- Deterministic operation and reconstructible systems.
- Kubernetes-native architecture compatible with GitOps workflows.
- Composable tools with focused operational responsibilities.

## Where it is going

Skaphos starts with concrete operational tools and evolves toward a cohesive operational system driven by real platform needs. The work is organized into two portfolios: Skaphos Platform for the core Kubernetes platform systems, and Adjacent Tools for complementary utilities such as Repokeeper and Wake.
